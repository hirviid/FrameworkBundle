HirviidFrameworkBundle
================

This bundle provides some abstract and base classes to use in a symfony2 project.


## Installation

    <?php
    // app/AppKernel.php

    public function registerBundles()
    {
        $bundles = array(
            // ...
            new Hirviid\Bundle\FrameworkBundle\HirviidFrameworkBundle(),
            // ...
        );
    }
